# React with TypeScript - Frontend

## dependencies

- React v17.0.2
  - `$ npm i react react-dom`
- TypeSCript v4.5.4
  - `$ npm i typescript`
  - `$ npm i @types/react @types/react-dom`
- ESLint
  - `$ npm i -D eslint`
- Prettier
  - `$ npm i -D prettier eslint-plugin-prettier eslint-config-prettier`
