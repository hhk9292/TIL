## 환경설정

### Backend

- Node.js v14
  - Express
  - Sequelize
- MySQL v8.0

### Frontend

- React

## 실행

- Backend

  1. `$ cd back`
  2. `$ npm install`
  3. `$ npx sequelize db:create` db 생성
  4. `$ npm run dev`
  5. `$ npx sequelize db:seed:all` 더미데이터 생성

- Frontend
  1. `$ cd front`
